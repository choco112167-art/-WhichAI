# WhichAI - AI 선택비서

상황만 입력하면 목적에 맞는 AI 도구를 추천해주는 작업 상황 분류 기반 AI 추천 서비스 (PoC).

## 1. 프로젝트 개요

- 사용자 자연어 입력을 간단한 규칙 기반 분류기로 작업 카테고리(회의/문서/이미지/영상/코드/PPT/리서치)로 분류.
- 카테고리별로 미리 정리한 대표 AI 도구 리스트에서 3~4개를 추천.
- 프론트엔드 없이, Colab/터미널에서 텍스트 입출력으로만 동작하는 프로토타입.

## 2. 실행 방법

1. 이 레포지토리를 클론하거나, `whichai_poc.py` 파일을 Colab에 업로드합니다.
2. Colab에서 해당 파일 내용을 하나의 코드 셀에 붙여 넣고 실행합니다.
3. 셀 실행 후, 프롬프트가 나오면 현재 하고 싶은 작업 상황을 한국어로 입력합니다.

예)
- `팀플 회의 녹음 요약해서 액션아이템 정리하고 싶어`
- `졸업논문 관련 논문 자료조사 좀 해야 돼`
- `유튜브 쇼츠 영상이랑 썸네일 같이 만들고 싶어`

## 3. 사용 예시 (콘솔 출력)

```bash
===== WhichAI - AI 선택비서 (규칙 기반 PoC) =====
현재 하고 싶은 작업 상황을 자연어로 입력하세요:
> 팀플 회의 녹음 요약해서 액션아이템 정리하고 싶어

[판단된 작업 카테고리]
- meeting_summary

[추천 AI 도구]
1. 클로바노트
   - 설명: 한국어 회의 녹음 자동 전사 및 요약 도구
   - 강점: 국내 회의 환경(한국어 음성)에 최적화

2. Notion AI
   - 설명: 노션 문서·회의록 요약 및 정리
   - 강점: 노션 워크스페이스와 자연스럽게 통합

3. Notta
   - 설명: 화상회의 전사 및 요약 도구
   - 강점: Zoom·Meet 등 주요 회의 플랫폼 지원
